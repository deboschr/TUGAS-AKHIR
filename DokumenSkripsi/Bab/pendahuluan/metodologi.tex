Metodologi penelitian yang digunakan pada tugas akhir ini adalah sebagai berikut:

\begin{enumerate}

    \item Melakukan studi literatur terhadap protokol HTTP, URI, HTML, dan \textit{web crawling}.

    \item Melakukan studi literatur dan eksperimen terhadap JavaFX, Jsoup, Java HTTP Client API, dan Apache POI.

    \item Menganalisis permasalahan dalam pemeriksaan tautan rusak pada situs web, meninjau perangkat lunak serupa, merumuskan kebutuhan fungsional dan non-fungsional, serta menganalisis teknologi yang digunakan dalam pengembangan perangkat lunak.

    \item Merancang kelas, alur, dan antarmuka pengguna.
    
    \item Mengimplementasikan perangkat lunak pemeriksa tautan rusak pada situs web.
    
    \item Menguji perangkat lunak yang telah dikembangkan dan membandingkan hasil pemeriksaannya dengan perangkat lunak serupa.
    
    \item Menulis dokumen tugas akhir.
    
\end{enumerate}
