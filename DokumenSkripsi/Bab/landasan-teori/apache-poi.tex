
Apache POI adalah pustaka Java \textit{open source} yang dikembangkan oleh Apache Software Foundation~\cite{apache:poi:docs}. Pustaka ini menyediakan API untuk memanipulasi berkas Microsoft Office dengan format yang berbasis OLE2 dan OOXML. OLE2 (\textit{Object Linking and Embedding} 2) adalah format berbasis biner yang digunakan sebelum Office~2007 dengan ekstensi seperti \texttt{.doc} untuk Word, \texttt{.xls} untuk Excel, dan \texttt{.ppt} untuk PowerPoint~\cite{microsoft:mscfbf}. Sementara itu, OOXML (\textit{Office Open} XML) adalah format berbasis XML yang diperkenalkan sejak Office~2007 dengan ekstensi seperti \texttt{.docx} untuk Word, \texttt{.xlsx} untuk Excel, dan \texttt{.pptx} untuk PowerPoint~\cite{ecma:376,iso:29500}.
