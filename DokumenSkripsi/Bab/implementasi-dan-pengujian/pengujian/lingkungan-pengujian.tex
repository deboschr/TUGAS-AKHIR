Lingkungan pengujian menjelaskan konfigurasi perangkat keras dan jaringan yang digunakan dalam proses pengujian aplikasi pemeriksa tautan rusak pada situ web. 

Berikut merupakan rincian lingkungan pengujian yang digunakan:
\begin{enumerate}
   \item \textbf{Lingkungan Perangkat Keras}\\
   Pengujian dilakukan menggunakan laptop pribadi dengan spesifikasi sebagai berikut:
   \begin{enumerate}
      \item Laptop: Lenovo Legion 5 (Model 82B5)
      \item \textit{Processor}: AMD Ryzen 5 4600H (12 \textit{Cores}, 16 \textit{Threads}), 3.0\,GHz
      \item RAM: 16\,GB DDR4
      \item Penyimpanan: SSD 512\,GB
      \item Sistem Operasi: Windows 11 64-bit
   \end{enumerate}

   \item \textbf{Lingkungan Jaringan}\\
   Seluruh pengujian dilakukan menggunakan hotspot ponsel sebagai sumber koneksi internet, dengan konfigurasi sebagai berikut:
   \begin{itemize}
      \item Jenis Jaringan: Hotspot ponsel (4G/LTE)
      \item Provider Internet: Telkomsel
      \item Rata-rata Kecepatan Unduh: 58{,}49\,Mbps
      \item Rata-rata Kecepatan Unggah: 46{,}63\,Mbps
      \item Rata-rata Latensi: 23\,ms
   \end{itemize}
\end{enumerate}
